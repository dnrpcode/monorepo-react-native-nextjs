## 📚 Shared Code
 - State Management: [Zustand](https://github.com/pmndrs/zustand)
 - VanillaJs

## Running the project?

🧪 **Init:**
```bash
  yarn
```

🌐 **Website:**
```bash
  yarn web:dev
```

🤖 **Android:**
```bash
  yarn mobile:android
```

🍏 **IOS:**
```bash
  cd apps/mobile
  bundle install
  cd ios
  pod install
  cd ../../../
  yarn mobile:ios
```
