
# 🚀 Monorepo Frontend - React Native and NextJS

This is project to learn sharing code JavaScript/TypeScript in monorepo use **Yarn Workspace**.

## 📚 Shared Code
 - State Management: [Zustand](https://github.com/pmndrs/zustand)
 - VanillaJs

## Running the project?

🧪 **Init:**
```bash
  yarn
```

🌐 **Website:**
```bash
  yarn web:dev
```

🤖 **Android:**
```bash
  yarn mobile:android
```

🍏 **IOS:**
```bash
  yarn mobile:ios
```
